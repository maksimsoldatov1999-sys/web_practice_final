Git

  • git init — создаёт новый репозиторий Git в текущей папке.
  • git add — добавляет изменения в индекс (staging area) для следующего коммита.
  • git commit — сохраняет изменения из индекса в репозиторий с комментарием.
  • git push — отправляет локальные коммиты в удалённый репозиторий.
  • git pull — забирает изменения из удалённого репозитория и сливает с текущей веткой.
